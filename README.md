# A3_Entrega

## Descrição do Projeto
O projeto A3_Entrega tem objetivo de garantir que o ambiente de desenvolvimento (VSCode, Git e JavaScript) está corretamente configurado, realizando um ciclo de trabalho Git completo: Criar Repositório > Clone > Alterar Ficheiros > Commit > Push.
